# Asset Rules

## Generated Assets
- Generate assets in reviewable batches.
- Normalize scale, anchors, and transparent padding before wiring into production.
- Keep source prompts next to approved output.
- Use screenshots to approve runtime fit.

## Naming
Use lowercase kebab-case for files and descriptive names for variants:

`wizard-run-approved-v1.png`

`spire-entry-tileset-v2.png`

## Runtime Checks
- Confirm alpha edges.
- Confirm mobile scale.
- Confirm color contrast against actual backgrounds.
